## Requirements

- [Rust](https://www.rust-lang.org/tools/install)
- [Cairo Lang](https://www.cairo-lang.org/docs/quickstart.html#quickstart): for class hash calculation.

## Run
  
```bash  
cargo run 
``` 

## To run while watching files
 
```bash
cargo watch -x run 
```
